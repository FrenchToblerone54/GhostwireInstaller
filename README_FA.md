# نصب‌کننده آسان گوست‌وایر

**[📖 English](README.md)**

یک نصب‌کننده گام‌به‌گام و کاربرپسند برای [گوست‌وایر](https://github.com/FrenchToblerone54/Ghostwire) — سیستم تونل معکوس ضد سانسور. این اسکریپت به‌صورت خودکار تشخیص می‌دهد که در ایران هستید یا خارج، و با توضیحات روشن در هر مرحله شما را راهنمایی می‌کند.

---

## گوست‌وایر چیست؟

گوست‌وایر یک تونل معکوس است که به کاربران داخل ایران امکان دسترسی آزاد به اینترنت می‌دهد.

```
[کاربر در ایران] → [سرور ایران] ←تونل WebSocket→ [کلاینت خارج] → [اینترنت]
```

- **سرور ایران** — روی پورت‌های محلی listen می‌کند، اتصال تونل را از کلاینت خارج می‌پذیرد
- **کلاینت خارج** — به سرور ایران متصل می‌شود و ترافیک را به اینترنت آزاد هدایت می‌کند

نکته کلیدی: ایران اتصالات _خروجی_ به سرورهای خارجی را مسدود می‌کند، اما همچنان می‌تواند اتصالات _ورودی_ دریافت کند. کلاینت خارج به‌صورت ورودی به ایران متصل می‌شود و تونل را ایجاد می‌کند.

---

## پیش‌نیازها

|                | سرور ایران                           | کلاینت خارج                                  |
| -------------- | ------------------------------------ | -------------------------------------------- |
| **مکان**       | VPS داخل ایران با IP عمومی           | VPS خارج از ایران (هلند، آلمان، آمریکا و...) |
| **سیستم‌عامل** | Linux x86_64 (Ubuntu 22.04 به بالا)  | Linux x86_64 (Ubuntu 22.04 به بالا)          |
| **دسترسی**     | `sudo` / root                        | `sudo` / root                                |
| **اختیاری**    | دامنه + Cloudflare برای پایداری بهتر | —                                            |

## شروع سریع

### مرحله ۱ — روی سرور ایران

```bash
wget https://raw.githubusercontent.com/FrenchToblerone54/GhostwireInstaller/main/setup.sh -O setup.sh
chmod +x setup.sh
sudo ./setup.sh
```

اسکریپت موقعیت شما را تشخیص می‌دهد، تأیید می‌کند که حالت **سرور ایران** می‌خواهید، سپس شما را در این مراحل راهنمایی می‌کند:

1. دانلود و تأیید فایل باینری سرور گوست‌وایر
2. تنظیم پورت WebSocket، نگاشت پورت‌ها، به‌روزرسانی خودکار و پنل وب اختیاری
3. نصب سرویس systemd تا گوست‌وایر به‌صورت خودکار شروع شود
4. راه‌اندازی اختیاری nginx با TLS از Let's Encrypt

**توکن احراز هویت نمایش داده‌شده در انتها را ذخیره کنید — برای کلاینت خارج لازم دارید.**

## سؤالات نصب (سرور ایران)

| سؤال               | پیش‌فرض            | توضیح                                                                                                              |
| ------------------ | ------------------ | ------------------------------------------------------------------------------------------------------------------ |
| WebSocket host     | `127.0.0.1`        | اگر از nginx استفاده می‌کنید از `127.0.0.1` استفاده کنید (توصیه‌شده). برای اتصال مستقیم از `0.0.0.0` استفاده کنید. |
| WebSocket port     | `8443`             | پورتی که کلاینت خارج به آن متصل می‌شود                                                                             |
| نگاشت پورت‌ها      | `8080=80,8443=443` | `پورت_ایران=پورت_اینترنت` — ترافیک ورودی به پورت*ایران به پورت*اینترنت در سمت خارج هدایت می‌شود                    |
| به‌روزرسانی خودکار | `Y`                | گوست‌وایر GitHub را برای به‌روزرسانی‌ها بررسی می‌کند و خودش ری‌استارت می‌شود                                       |
| پنل وب             | `Y`                | داشبورد مرورگری برای مانیتورینگ و کنترل                                                                            |

**مثال‌های نگاشت پورت:**

```
8080=80          پورت ایران 8080 ← پورت اینترنت 80
8443=443         پورت ایران 8443 ← پورت اینترنت 443
9000=1.1.1.1:53  پورت ایران 9000 ← 1.1.1.1:53
8000-8100=3000   فورواردینگ محدوده پورت
```

## بعد از نصب

### دستورات مفید

**سرور ایران:**

```bash
sudo systemctl status ghostwire-server
sudo systemctl restart ghostwire-server
sudo journalctl -u ghostwire-server -f
sudo ghostwire-server update
```

**کلاینت خارج:**

```bash
sudo systemctl status ghostwire-client
sudo systemctl restart ghostwire-client
sudo journalctl -u ghostwire-client -f
sudo ghostwire-client update
```

### فایل‌های تنظیمات

- سرور ایران: `/etc/ghostwire/server.toml`
- کلاینت خارج: `/etc/ghostwire/client.toml`

برای دیدن توکن در آینده:

```bash
grep token /etc/ghostwire/server.toml
```

## نکات Cloudflare

اگر از Cloudflare جلوی دامنه سرور ایران استفاده می‌کنید:

- **Network → WebSockets**: باید **روشن** باشد
- **SSL/TLS → Overview**: روی **Full (Strict)** تنظیم کنید
- **Speed → Rocket Loader**: **خاموش** کنید
- **Speed → Auto Minify**: همه را غیرفعال کنید

روی کلاینت خارج، `/etc/ghostwire/client.toml` را ویرایش کنید و تنظیم کنید:

```toml
[cloudflare]
enabled=true
max_connection_time=1740
```
